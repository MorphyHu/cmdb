**CMDB**

    使用python的 django 框架研发，所用python 版本 3.4, django 版本 1.10.6
    

**安装/运行**

    1. 安装 python >= 3.4
    
    2. git clone https://github.com/charlesxs/cmdb.git && cd cmdb
    
    3. pip install -r requirements.txt
    
    4. python manage.py runserver 0.0.0.0:8000
